## Описание лабораторных работ

- **`lab1.py`: Обработка изображений**
  - Загрузка изображения (`perec.jpg`).
  - Применение размытия по Гауссу.
  - Повышение резкости двумя методами: свертка с жестко заданным ядром и метод маски нерезкости.
  - Выделение границ с помощью оператора Собеля.
  - Комбинирование различных обработанных версий изображения.
  - Визуализация оригинального, промежуточных и финального комбинированного изображений с использованием `matplotlib`.

- **`lab2.py`: Отслеживание объекта по цвету в видео**
  - Захват видеопотока из файла (`catball.mp4`).
  - Преобразование кадров в цветовое пространство HSV.
  - Создание бинарной маски для заданного диапазона цвета (в коде оттенки от красного до желтого).
  - Применение морфологических операций (эрозия, дилатация) к маске.
  - Поиск и фильтрация контуров на маске по площади.
  - Определение центра масс самого большого подходящего объекта.
  - Визуализация обнаруженного объекта (контур, центр) на кадре.
  - Возможность остановки воспроизведения по нажатию клавиши 'q'.

- **`lab3.py`: Обнаружение лиц, глаз и улыбок в видео**
  - Захват видеопотока из файла (`Every winner from the HLTV Awards 2024.mp4`).
  - Использование предобученных каскадов Хаара (`haarcascade_...`) для обнаружения лиц, глаз и улыбок.
  - Отображение прямоугольников вокруг обнаруженных элементов.
  - Вывод текста "SMILE" или "OPEN EYES", если соответствующий элемент не обнаружен на лице.
  - Расчет и отображение FPS (кадров в секунду).
  - Возможность остановки воспроизведения по нажатию клавиши 'q'.

- **`lab_1.py`: Калибровка камеры**
  - Загрузка набора изображений шахматной доски из папки `images/`.
  - Автоматическое определение углов шахматной доски на изображениях.
  - Вычисление параметров камеры (матрицы камеры `mtx`, коэффициентов искажения `dist` и т.д.) с помощью `cv2.calibrateCamera`.
  - Вывод рассчитанной матрицы камеры.

- **`main.py`: Анализ объектов на изображении**
  - Загрузка изображения (`photo.jpg`).
  - Преобразование в градации серого.
  - Бинаризация изображения с помощью пороговой обработки.
  - Поиск контуров на бинарном изображении.
  - Подсчет количества найденных объектов (контуров).
  - Вычисление площади и координат центра масс для каждого объекта.
  - Определение самого большого и самого маленького объекта по площади.
  - Визуализация: отображение площадей, пометка центров особых объектов, рисование контуров на копии исходного изображения.
